SEÇÃO 3: PEDIDOS (ms-pedidos, porta 8080) + PAGAMENTOS (ms-pagamentos, 8081)
================================================================================

3.1) Criar um pedido (end-to-end com pagamento):

    Execute em PowerShell:

    $clienteId = [guid]::NewGuid().ToString()
    Write-Host "Cliente ID para este teste: $clienteId"

    $pedidoBody = @{
      clienteId = $clienteId
      restauranteId = "a1b2c3d4-0000-0000-0000-000000000001"
      itens = @(
        @{
          itemId = "c1000000-0000-0000-0000-000000000001"
          nomeItem = "Picanha na brasa"
          precoUnitario = 89.90
          quantidade = 1
        }
      )
      observacao = "Sem cebola, por favor"
      enderecoEntrega = "Rua Teste, 100, Centro"
    } | ConvertTo-Json -Depth 6

    $pedidoResp = Invoke-RestMethod -Method Post -Uri "http://localhost:8080/v1/pedidos" `
      -ContentType "application/json" -Body $pedidoBody

    $pedidoId = $pedidoResp.id
    Write-Host "Pedido criado: $pedidoId"
    $pedidoResp | ConvertTo-Json -Depth 6

    Esperado: HTTP 201 (Created) com JSON contendo:
      - id (UUID válido)
      - clienteId (igual ao enviado)
      - restauranteId = "a1b2c3d4-0000-0000-0000-000000000001"
      - status = "PENDENTE"
      - valorTotal = 89.90

    Status: [ ] PASS  [ ] FAIL


3.2) Consultar pedido criado:

    Invoke-RestMethod -Method Get -Uri "http://localhost:8080/v1/pedidos/$pedidoId" | ConvertTo-Json -Depth 6

    Esperado: HTTP 200 com JSON retornando o pedido com id=$pedidoId

    Status: [ ] PASS  [ ] FAIL


3.3) Processar pagamento para o pedido:

    Execute em PowerShell (usando $pedidoId e $clienteId do passo anterior):

    $pagBody = @{
      pedidoId = $pedidoId
      clienteId = $clienteId
      valor = 89.90
